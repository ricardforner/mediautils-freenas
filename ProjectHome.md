## Overview ##
The goal of this project is create a set of tools for your media files in FreeNAS, like organize transmission downloads, etc.

## Status ##
Feb 26, 2012 Add source code 0.1.0.

## Requirements ##

  * A FreeNAS BASE OS <= 0.7.x (WebGUI PHP based)
